## Методы примитивов

JS создает специальную объект-обертку для вызова методов примитива. Этот процесс очень оптимизированный.

Почему `new Number, String` etc. опасен:

```javascript
const value = new Boolean(false);

if(value) // do smth
```

Условие `if` будет `true`, потому что `typeof value = object` -> `({}) = true`.

### Числа (number)

JS хранит числа в 64 битном формате: 52 для цифр, 11 для положения точки, и последний бит для знака.

```javascript
alert(1e500); // Infinity
```

В JS часто встречается ошибка с потерью точности. Это из-за хранения чисел в бинарном формате, и *дроби* хранятся как бесконечные последовательность - `0.1 -> 0.0001100110011001100110011001100110011001100110011001101`. Поэтому:

```javascript
let sum = 0.1 + 0.2;
console.log(sum); // 0.30000000000000004
```

`num.toString(base)` - строковое представление `num` в системе счисления `base`.

```javascript
let num = 20;

console.log(num.toString(2)); // 10010
console.log(20..toString(8)); // 24
```

После числа ставим двойную точку, потому что программа ожидает плавающую точку.
`base` варьируется от 2 до 36. По умолчанию 10.

#### Округление

1. `Math.floor` - округление в меньшую сторону. 2.5 -> 2, -2.5 -> -3.
2. `Math.ceil` - округление в большую сторону. 2.5 -> 3, -2.5 -> -2.
3. `Math.round` - округление до ближайшего целового. 2.5 -> 3, -2.4 -> -2.
4. `Math.trunc` (в IE нет поддержки) - удаляет дробную часть. 2.5 -> 2.

`num.toFixed(n)` - округление числа до n знаков после запятой, возвращает строку.

```javascript
const num = 1.5246;
console.log(num.toFixed(2)); // "1.52"
```

`parseInt(str, radix)` - возвращает число в строке `str`. `radix` опциональный, используется для системы счисления. Останавливается, если видит *не число*.
`parseFloat(str)` - возвращает число с плавающей точкой.

#### Проверка чисел

- `isNaN` превращает значение в число, потом проверяет на `NaN`. Более строгая версия без преобразования в число - `Number.isNaN`.
- `isFinite` превращает значение в число, и возвращает `true`, если число обычное, а не `NaN/Infinity/-Infinity`. Более строгая версия без преобразования в число - `Number.isFinite`.
- `Object.is` - функционал как у `===` (строгое равенство), но 2 различий:
    1. `Object.is(NaN, NaN)` - `true`.
    2. `Object.is(0, -0)` - `true`.

#### Другие мат. функции

`Math.random()` - псевдослучайное число от 0 до 1. Пример: 0.01234.
`Math.min() & Math.max()` - возвращает min и max число из перечисленных аргументов соответственно.
`Math.pow(n, power)` - возвращает число n, возведённое в степень power.

### Строки (string)

`\` - символ экранирования (backslash). **Спецсимволы**:

- `\n` - перевод строки.
- `\t` - знак табуляции.
- `\b`, `\f`, `\v` - Backspace, Form Feed и Vertical Tab - не используются.

`str.length` - длина строки.

#### Доступ к символам

1. `str[pos]` - символ в позиции `pos`, начинаем от нуля. Отрицательный `pos` - `undefined`.
2. `str.at[pos]` - тоже самое, но `pos` может быть отрицательным числом. `pos = -1` последний символ.

`for...of` помогает перебрать строку:

```javascript
for (let char of "Dias") {
    console.log(char); // "D", "i", "a", "s"
}
```

#### Изменение регистра

1. `toLowerCase()` - нижний регистр.
2. `toUpperCase()` - верхний регистр.

#### Поиск подстроки

- `str.indexOf(substr, pos)` - ищет `substr` в `str` начиная с позиции `pos`. Возвращает позицию, если нет совпадении - `-1`.

Проверка:

```javascript
if (str.indexOf(substr) != -1) return true;

if (~str.indexOf(substr)) return true;
```

- `str.lastIndexOf(substr, position)` - ищет с конца строки.

- `str.includes(substr, pos)` - это современный метод, возвращает `true || false`.

- `str.startsWith(substr, pos)` - начинается ли строка с `substr`.

- `str.endsWith(substr, pos)` - заканчивается ли строка с `substr`.

#### Получение подстроки

- `str.slice(start, [, end]` - возвращает строки от `start` до (не включая) `end`.

```javascript
let str = "Diastan";

console.log(str.slice(1)); // iastan

console.log(str.slice(1, 3)); // ia

console.log(str.slice(-3, -1)); // ta

console.log(str.slice(3, 1)); // ""
```

- `str.substring(start, [, end])` - возвращает строки от `start` до (не включая) `end`. Не поддерживает отрицательные позиции (принимает их за `0`), но если `end` больше `start` - поменяет местами.

```javascript
let str = "Dagistan";

console.log(str.substring(1, 4)); // agi
console.log(str.substring(4, 1)); // age
```

- `str.substr(start, [, length])` - возвращает часть строки от `start` длины `length`.

```javascript
let str = "Indistan";

console.log(str.substr(0, 3)); // Ind
```

#### Операторы сравнения

Правила:

1. `"a" > "Z"` - строчные буквы больше заглавных.
2. `"Österreich" > "Zealand"` - буквы с диакратическими знаками идет *не по порядку*.

`str.codePointAt(pos)` - возвращает код символа (UTF-16).
`String.fromCodePoint(code)` - создает символ по коду.

#### Правильное сравнение

`string.localeCompare(compareString, locales, options)` - сравнение строк. Возвращает:

- отрицательное число, если `string` меньше `compareString`.
- положительное число, если `string` больше `compareString`.
- `0`, если равны.

#### Другие методы

`trim()` - убирает пробелы в начале и конце строки. Есть доп - `trimStart() & trimEnd()`.
`split(div, n)` - разбивает строку на массив, `n` - ограничивает.

```javascript
const str1 = "D s";
const str2 = "D,s";
const str3 = "";

str1.split(""); // ["D", " ", "s"]
str1.split(" "); // ["D", "s"]
str1.split(","); // ["D s"]
str2.split(","); // ["D", "s"]
str3.split(","); // [""]
str3.split(""); // []
```

`replace(str1, str2)` - меняет первое совпадание, `str1` на `str2`.
`replaceAll(str1, str2)` - меняет все совпадения. В обоих случаях, если совпадении нет, возвращает строку.
