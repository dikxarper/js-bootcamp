## Методы массивов

### slice

`arr.slice([start], [end])` - возвращает новый массив, копируя элементы.

```javascript
let arr = [1, 2, 3, 4];
let copy = arr.slice(); // для копирования массива

console.log(arr.slice(1, 3)); // [2, 3]
console.log(copy); // [1, 2, 3, 4]
```

### concat

`arr.concat(arg1, arg2)` - создает массив, который включает в себя данные `arg1, arg2, etc`.

```javascript
let arr1 = [1, 2];

console.log(arr1.concat(3, 4)); // [1, 2, 3, 4]
console.log(arr1.concat([3], 4)); // [1, 2, 3, 4]

let arr2 = {
    0: "wtf",
    1: "wth",
};

console.log(arr1.concat(arr2)); // [1, 2, [object Object]]

let arr3 = {
    0: "wtf",
    1: "wth",
    [Symbol.isConcatSpreadable]: true,
    length: 2,
};

console.log(arr1.concat(arr3)); // [1, 2, "wtf", "wth"]
```

### forEach

`arr.forEach(function(item, index, array) {})` - запускает функцию для каждого элемента массива, возвращает `undefined`. Функция вызывает ради действия.

```javascript
let arr = [1, 2, 3];

arr.forEach((elem) => {
    console.log(elem); // 1 2 3
});
```

### indexOf/lastIndexOf и includes

`arr.indexOf(item, from)` - ищет `item` начиная с индекса `from` - возвращает индекс, в противном случае - `-1`.
`arr.lastIndexOf(item, from)` - ищет `item` начиная с индекса `from` с конца (справа налево) - возвращает индекс, в противном случае - `-1`.
`arr.includes(item, from)` - ищет `item` начиная с индекса `from` - возвращает true, в противном случае - `false`.

```javascript
let arr = [1, 2, 3, 4];

console.log(arr.indexOf(2, 2)); // -1
console.log(arr.lastIndexOf(2, 2,)); // 1
console.log(arr.includes(2)); // true
```

`includes()` правильно обрабатывает `NaN`:

```javascript
let arr = [NaN];

console.log(arr.indexOf(NaN)); // -1, неверно
console.log(arr.includes(NaN)); // true
```

### find и findIndex/findLastIndex

`arr.find(function(item, index, array) {})` - используется для массивов объектов, возвращает найденный объект.
`arr.findIndex(function(item, index, array) {})` - используется для массивов объектов, возвращает индекс найденного объекта.
`arr.findLastIndex(function(item, index, array) {})` - используется для массивов объектов, возвращает индекс найденного объекта справа налево.

```javascript
let users = [
    {id: 1, name: "Sara"},
    {id: 2, name: "Aira"},
    {id: 3, name: "Saira"},
    {id: 4, name: "Aira"},
];

let user1 = users.find(el => el.id === 1);
let user2 = users.findIndex(el => el.name === "Aira");
let user3 = users.findLastIndex(el => el.name === "Aira");

console.log(user1); // {id: 1, name: "Sara"}
console.log(user2); // 1
console.log(user3); // 3
```

### filter

`arr.filter = (function(item, index, array) {})` - ищет элементы, которые возвращают true на условие.

```javascript
let users = [
    {id: 1, isEmployed: true},
    {id: 2, isEmployed: false},
    {id: 3, isEmployed: false},
    {id: 4, isEmployed: true},
];

let employedUsers = users.filter(user => user.isEmployed === true);

console.log(employedUsers); // [{id: 1, isEmpoyed: true}, {id: 2, isEmployed: true}]
```

### map

`arr.map(function(item, index, array) {})` - вызывает функцию для каждого элемента, возвращает массив из новых значении.

```javascript
let users = [
    {id: 1, isEmployed: true},
    {id: 2, isEmployed: false},
    {id: 3, isEmployed: false},
    {id: 4, isEmployed: true},
];

let doubleIds = users.map(el => el.id * 2);
console.log(doubleIds); // [2, 4, 6, 8]
```

### sort

`arr.sort(fn)` - сортирует масссив на месте, меняя в нём порядок элементов. Элементы сортируются как строки.
Как именно работает функция сортировки:

- return > 0, a ставится после b.
- return 0, порядок не меняется.
- return < 0, a ставится перед b.

```javascript
let arr = [ 100, 12, 84 ];

arr.sort(function(a, b) { 
    return a - b; 
});

arr.sort((a - b) => a - b); // тоже самое

console.log(arr); // [12, 84, 100], простая сортировка по возрастанию
```

### reverse

`arr.reverse()` - меняет порядок на обратный.

```javascript
let arr = [1, 2, 3];

console.log(arr.reverse()); // [3, 2, 1]
```

### split и join

`str.split(delim, n)` - возвращает массив после разделения с `delim`, и можно ограничить число возвращаемых элементов с `n`.

```javascript
let str = "wtf";

console.log(str.split("")); // ["w", "t", "f"]
```

`arr.join(glue)` - возвращает строку, объединив элементов массив через `glue`.

```javascript
let arr = [1, 2, 3];

console.log(arr.join(" ")); // "1 2 3"
```

### reduce/reduceRight

`arr.reduce(function(accumulator, item, index, array) {}, [initial])` - используется для вычисления единого значения на основе всего массива. `accumulator` - результат предыдущего вызова функции, равен `inital` при первом вызове.

```javascript

let arr = [1, 2, 3, 4];

console.log(arr.reduce((sum, curr) => sum + curr, 0)); // 1 + 2 + 3 + 4 = 10
console.log(arr.reduce((sum, curr) => sum + curr)); // 10, но если arr = [], то выдаст ошибку без initial
```

`arr.reduceRight` - работает аналогично, но проходит справа налево.

### Array.isArray

`Array.isArray(arr)` - проверяет массив или нет.

```javascript
let arr = [1, 2];

console.log(typeof arr); // object
console.log(Array.isArray(arr)); // true
```

### thisArg

`arr.method(func, thisArg)` - параметр, значение которого становится `this`. Все методы кроме `arr.sort()` принимает `thisArg`.

```javascript
let users = [
    {id: 1, age: 18},
    {id: 2, age: 8},
    {id: 3, age: 28},
    {id: 4, age: 84},
];

const ageStat = {
    youngAgeMin: 18,
    youngAgeMax: 50,
    determineYoung(user) {
        return user.age >= this.youngAgeMin && user.age <= this.youngAgeMax;
    },
};

let youngUser = users.filter(ageStat.determineYoung, ageStat);
let youngUser = users.filter(user => ageStat.determineYoung(user)); // тоже самое

console.log(youngUser[0].age); // 18 
console.log(youngUser[1].age); // 28
```

### some и every

`arr.some(fn)` - хоты бы один элемент подходит под условие - `true`.
`arr.every(fn)` - все элементы подходят под условие - `false`.

```javascript
let arr = [
    {id: 1, isStudent: true},
    {id: 3, isStudent: false},
    {id: 5, isStudent: true},
];

console.log(arr.some(value => value.isStudent === true)); // true
console.log(arr.some(value => (value % 2) === 1)); // true
```
