## Массивы

**Массив** (Array) используется для хранения упорядоченных коллекций.
Создание пустого массива:

```javascript
let arr = new Array();
let arr = [];
```

Последний элемент массива можно получить так - `arr.at[-1]`.

JavaScript поддерживает **Queue** и **Stack** (двусторонняя очередь).

> [!NOTE] Queue и Stack
> Методы массива позволяют использовать его как разные структуры данных.
>
> **Queue (очередь, FIFO — *First In, First Out*)**
> Первый добавленный элемент выходит первым.
>
> - `push()` — добавить элемент в конец.
> - `shift()` — удалить первый элемент.
>
> **Stack (стек, LIFO — *Last In, First Out*)**
> Последний добавленный элемент выходит первым.
>
> - `push()` — добавить элемент в конец.
> - `pop()` — удалить последний элемент.

### Методы массивов

Массив работае с методами:

1. `push()` - возвращает новую длину.
2. `shift()` - возвращает удаленный элемент.
3. `unshift()` - возвращает новую длину.
4. `pop()` - возвращает удаленный элемент.

### Перебор элементов массива

У нас имеется массив - `let arr = [1, 2, 3]`.

- `for`

```javascript
for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}
```

- `for...of` - более чем достаточно.

```javascript
for (let num of arr) {
    console.log(num);
}
```

- `for...in` - можно так, потому что массив `=` объект.

```javascript
for (let key in arr) {
    console.log(arr[key]);
}
```

Последнее свойство лучше не использовать под массивы. Есть 2 причины:

- `for...in` выведет все свойства массива (объекта). `for...of` выведет только свойства массива с цифровыми индексами ([0], [1]...).
- Он работает 10-100 раз медленнее чем `for...of`.

**new Array()** - создает массив, но есть одна хитрость.

```javascript
let arr = new Array(5);

console.log(arr[0]); // undefined
console.log(arr.length); // 5
```

> `length` = arr[max] + 1. И мы можем вручную менять `length`. Например, для очистки массива `arr.length = 0`.

### toString и сравнения

У массивов нет `Symbol.toPrimitive()` и `valueOf()`. При преобразовании они всегда используют `toString()`.

```javascript
let arr = [1, 2, 3];

console.log(arr == "1, 2, 3"); // true
console.log(arr ==== "1, 2, 3"); // false
console.log(arr + "1"); // "1,2,31"
```

Не сравнивайте массиы с `==`.

```javascript
console.log([] == []); // false
console.log([1] == [1]); // false

alert(0 == []); // true, [] -> "" -> 0
alert("0" == []); // false, [] -> ""
```

### splice

`arr.splice(start[, deleteCount, elem1, ..., elemN])` - удаляет, добавляет и заменяет. Возвращает удаленное значение.

```javascript
let animals = ["rabbit", "fox", "wolf"];
let colors = ["blue", "red", "yellow"];
let browsers = ["firefox", "chrome"];

animals.splice(0, 2, "bear"); // ["rabbit", "fox"], удалили и добавили
colors.splice(2, 1); // ["yellow"], удалили
browsers.splice(1, 0, "edge"); // [], добавили

console.log(animals); // ["bear", "wolf"]
console.log(colors); // ["blue", "red"]
console.log(browsers); // ["firefox", "chrome", "edge"]
```
