# Data Types

В JavaScript существует восемь типов данных:

- семь примитивных;
- один непримитивный (`Object`).

---

## Primitive Types

### Number

`Number` представляет целые числа и числа с плавающей точкой.

Специальные значения:

- `Infinity` — например, результат деления положительного числа на ноль.
- `-Infinity` — например, результат деления отрицательного числа на ноль.
- `NaN` — специальное значение типа `number`, означающее, что результат вычисления не может быть представлен числом.

Интересный факт:

```javascript
typeof NaN === "number";
```

Любое сравнение с `NaN` возвращает `false`, даже:

```javascript
NaN === NaN;
```

---

### BigInt

`BigInt` представляет целые числа произвольной длины.

Безопасный диапазон `Number`:

```javascript
-(2 ** 53 - 1)
```

до

```javascript
2 ** 53 - 1
```

Чтобы создать `BigInt`, добавьте `n`:

```javascript
123n
```

Арифметические операции между `BigInt` и `Number` вызывают `TypeError`.

---

### String

`String` представляет текст.

Строку можно создать:

- одинарными кавычками;
- двойными кавычками;
- обратными кавычками.

Шаблонные строки поддерживают выражения:

```javascript
`${expression}`
```

---

### Boolean

Имеет два возможных значения:

- `true`;
- `false`.

---

### null

`null` — специальное пустое значение, которое программист задаёт намеренно.

Несмотря на то что

```javascript
typeof null === "object"
```

это историческая ошибка JavaScript.

`null` является примитивом.

---

### undefined

Обычно означает, что значение ещё не было присвоено. Можно и вручную присвоить, но не рекомендуется.

---

### Symbol

`Symbol` представляет уникальное примитивное значение.

Чаще всего используется как ключ свойства объекта.

Свойства с ключами `Symbol` игнорируются:

- `for...in`;
- `Object.keys()`;
- `JSON.stringify()`.

---

## Non-Primitive Types

### Object

`Object` представляет коллекции данных и более сложные сущности.

Массивы и функции тоже являются объектами. Массив `Array` является специализированным объектом.

Для функций:

```javascript
typeof function () {} === "function";
```

Хотя функция остаётся объектом.
