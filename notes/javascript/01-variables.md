# Variables

## Что такое переменная

Переменная — именованное хранилище данных.

Рекомендуется объявлять и инициализировать переменную в одной строке. Разделение объявления и присваивания может ухудшать читаемость кода.

```javascript
let message = "Hello";
```

---

### let

`let` позволяет изменять значение переменной во время выполнения программы.

```javascript
let text = "What the fuck?";

text = "What the fock?";

console.log(text); // What the fock?
```

Переменная, объявленная через `let`:

- может быть переопределена;
- имеет блочную область видимости (block scope).

---

## const

Переменная, объявленная через `const`:

- не может быть переопределена;
- если хранит объект или массив, их содержимое можно изменять.

```javascript
const user = {
    name: "Dias",
};

user.name = "Alex"; // допустимо
```

---

### var

Переменная, объявленная через `var`:

- инициализируется значением `undefined`;
- имеет функциональную область видимости (function scope), а не блочную.

**Hoisting (поднятие)** означает, что объявления обрабатываются до выполнения кода, даже если они находятся ниже в исходном коде.

---

## Имена переменных

Имена переменных чувствительны к регистру.

Допустимые символы:

- буквы;
- цифры;
- `_`;
- `$`.

Имя переменной не может начинаться с цифры.

---

## Типизация JavaScript

JavaScript — язык с динамической и слабой типизацией.

При создании переменной тип данных указывать не нужно.
