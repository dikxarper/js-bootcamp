# Небольшой конспект по 2 дню обучения

## Значение

Значение - данные, с которыми работает программа. Тип данных принадлежит значению, а не переменной. Поэтому одной переменной можно присваивать значения разных типов.

`null` - специальное пустое значение, которое задается программистом намеренно.

Типы данных нужны программе, чтобы понимать, как хранить значения и какие операции с ними можно выполнять. Типы данных не меняются, а значения меняются, и вместе с ним меняются типы значении.

## Преобразование

`NaN (Not a Number)` - специальное значение типа number, означающее, что результат вычисления не может быть представлен числом. Любое сравнение с `NaN` возвращает `false`, даже `NaN === NaN`.
`Number.isNaN("value")` - false, потому что `"value"` является строкой, а не `NaN`.
`isNaN(value)` сначала выполняет `Number(value)`, затем проверяет, является ли результат `NaN`.
`parseInt()` - начинает читать строку слева направо. Если первый значимый символ является началом числа, то считывает до первого неподходящего символа. Иначе возвращает `NaN`:

- parseInt("a1") // NaN
- parseInt("")   // NaN

Преобразовывания:

1. Если одна из операндов является строкой и оператор "+", то выполняется конкатенация. Оба операнда преобразуются к строке, и результат - последовательность символом операндов по порядку.
2. Все остальные операторы ("-", "*", "/", "%", "**") преобразовывают операнды в тип данных `number`, и выполняется математическое выражение. `null` - 0, `undefined` - NaN, `true` - 1, `false` - 0.

Falsy:

- Boolean(0)         // false
- Boolean(NaN)       // false
- Boolean(null)      // false
- Boolean(undefined) // false
- Boolean("")        // false

Truthy (Любое значение, не являющееся falsy, является truthy):

- Boolean([])
- Boolean({})
- Boolean(function() {})

JavaScript не смотрит на смысл значения. Он применяет правила преобразования для конкретного типа данных.

## Базовые операторы и математика

Оператор является *бинарным*, если количество операндов - 2. Пример: 1 + 2.
*Унарным* - если количество 1. Пример: +1.

- +value это и есть короткая форма Number(value).
- -value = -(Number(value)) = -number

** - оператор считает справа налево. То есть:
`2 ** 3 ** 2 = 2 ** (3 ** 2) = 2 ** 9 = 512`

Оператор остатка, примеры:

- `8 % 2 = 0`
- `-8 % 2 = -0`
- `8 % 10 = 8`

Знак результата совпадает со знаком первого операнда.

### Инкремент и декримент

- Префиксный инкремент - `++a`: увеличивает `a` на `1` -> возвращает новое значение.
- Постфиксный инкремент - `a++`: запоминает старое значение -> увеличивает `a` на `1` -> возвращает старое значение.

Сначала изменить → потом вернуть значение.

- Префиксный декримент - `--a`: уменьшает  `a` на `1` -> возвращает новое значение.
- Постфиксный декримент - `a--`: запоминает старое значение -> уменьшает  `a` на `1` -> возвращает старое значение.

Сначала вернуть значение → потом изменить.
