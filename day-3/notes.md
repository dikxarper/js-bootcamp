# Операторы сравнения

Оператор `,` выводит последнее значение. Например:

`let result = (2, 3, 1) = 1`

## По теме

При сравнении строк:

1. Сравниваются коды первых символов строк.
2. Код заглавных букв ниже чем у строчных.
3. Если одна строка является префиксом другой (например "ab" и "abc"), то более длинная строка считается больше.

Сравнения:

1. `==` - нестрогое сравнение. Использует специальный алгоритм сравнения и при необходимости выполняет преобразование типов.
2. `===` - строгое сравнение. Сравниваются значения и тип данных без преобразования.

`null == undefined` -> true. `null` равен только `undefined`, и самому себе. А вот операторы сравнения преобразуют `null` в число 0.

`undefined` тоже равен только `null`, и самому себе (при `==`).

## Логические операторы

`||` -> или:

- true `||` true = true
- true `||` false = true
- false `||` true = true
- false `||` false = false

Короткое замыкание (short curcuit) - вычисление прекращается когда видит:

- truthy value для `||`
- falsy value для `&&`

`&&` -> и:

- true `&&` true = true
- true `&&` false = false
- false `&&` true = false
- false `&&` false = false

`!!value` -> `Boolean(value)`

Оператор нулевого слияния (Nullish coalescing) `??` - использует правое значение только если левое `null || undefined`.

JavaScript запрещает смешивать `??` с `||` и `&&` без явных скобок.

Приоритет:

1. `!`
2. `&&`
3. `||`

## Цикл

Разница между `while` и `do...while` проявляется только тогда, когда условие изначально `false`.

- `break` полностью завершает цикл;
- `continue` пропускает только текущую итерацию и переходит к следующей.

`break` и `continue` всегда завершает только ближайший цикл, в котором он находится.
