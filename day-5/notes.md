## Object

Объект - набор данных с ключами (свойствами) и значениями.

Если вызвать свойство объекта, которого нет - `undefined`;
`in` возвращает `true` если свойство есть, если нет - `false`;

Прямое равенство не копирует объект, а копирует ссылку на объект:

```python
const obj1 = {
    key: "value1",
};
const obj2 = obj1;
obj.key = "value2";

console.log(obj1.key); // "value2";
console.log(obj2.key); // "value2";
```

Для копирования объекта использовать:

1. Оператор расширения (spread operator) `...`. Это поверхнестное копирование (shallow copy): `const copy = { ...obj }`
2. Метод `structuredClone()`. Тут уже глубокое копирование (deep copy). Но функции вызовут ошибку.

## Garbage Collection

``` python
const object = {
    name: "Dias"
};

object = null;
```

В этом случае объект не удаляется, а на него больше никто не указывает. Поэтому он становится *недостижимым* (unreachable). Движок JS сам когда-нибудь удалит его из памяти.
Если объект все еще *достижим* (reachable), то на него ссылаются ссылки.

## Методы объекта и `this`

Метод — это функция, которая хранится в свойстве объекта.

`this` - это специальная значение, которое JS определяет в момент вызова функции.
В строгом режиме: `this === undefined`.

```python
const user = {
    name: "Dias",
    funct() {
        console.log(this.name);
    }
};

user.funct(); // "Dias"
(user.funct)(); // "Dias"
```

Скобки `(user.funct)()` - не меняют ничего. `(user.funct)() === user.funct()`.

Анонимная функция - функция без идентификатора, которую можно вызывать на месте.

Объект перед точкой определяет this только для обычной функции. Для стрелочной — нет. Стрелочная функция запоминает внешнее лексическое выражение.
